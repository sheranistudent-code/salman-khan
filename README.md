یہ ایک جدید اور ریسپانسیو (Responsive) Dark Mode UI ہے جو خاص طور پر AI ٹولز، وائس اسسٹنٹس، یا منی ایپلیکیشنز کے لیے ڈیزائن کیا گیا ہے۔ اس میں CSS ویری ایبلز، اینیمیشنز، اور گلاس مورفزم (Glassmorphism) کا استعمال کیا گیا ہے۔

✨ نمایاں خصوصیات (Features)
Modern Dark UI: گہرے نیلے اور کالے رنگوں کا خوبصورت امتزاج۔

Glassmorphism: شیشے جیسا شفاف اثر (Backdrop blur)۔

Animated Mic Section: مائیکروفون کے لیے پلس اینیمیشن (Pulse Animation)۔

Side Panels: ہسٹری اور سیٹنگز کے لیے سلائیڈنگ پینلز۔

Custom Form Elements: خوبصورت ڈیزائن والے سوئچ بٹنز اور ٹیکسٹ ایریا۔

Performance: ہلکا پھلکا کوڈ اور تیز لوڈنگ۔

🛠️ استعمال شدہ ٹیکنالوجیز (Technologies)
HTML5 (Structure)

CSS3 (Styling & Animations)

Custom Fonts: Segoe UI

📸 ڈیزائن کے اجزاء (UI Components)
اس کوڈ میں درج ذیل اہم اجزاء شامل ہیں:

Header: لوگو اور ایکٹیو اسٹیٹس بیج کے ساتھ۔

Glass Box: مین ان پٹ ایریا جہاں ٹیکسٹ لکھا جاتا ہے۔

Mic Circle: وائس کمانڈ کے لیے اینیمیٹڈ بٹن۔

Action Grid: فوری ایکشنز کے لیے بٹنز کا مجموعہ۔

Side Panel: ہسٹری دیکھنے کے لیے چھپا ہوا پینل۔

🚀 انسٹالیشن (Usage)
اس ڈیزائن کو اپنے پراجیکٹ میں استعمال کرنے کے لیے:

اس کوڈ کو اپنی style.css فائل میں کاپی کریں۔

اپنے HTML فائل میں ان کلاسز (Classes) کا استعمال کریں جو CSS میں دی گئی ہیں، جیسے کہ .app-wrap, .glass-box, وغیرہ۔
